# Project Overview

## Background
누리맵은 누리미디어 구성원이 직접 가본 장소 정보를 함께 축적하는 내부 지도 서비스다.

현재 일반 지도 앱은 제로페이 가맹점 여부를 원하는 방식으로 필터링하기 어렵다. 
그래서 구성원이 네이버 지도 URL을 기반으로 장소를 등록하고, 제로페이 가능 여부와 장소 정보를 집단 지성으로 쌓는 것이 이 프로젝트의 출발점이다.

## Goal
이 프로젝트의 목표는 사내 구성원이 네이버 지도 URL만으로 장소를 쉽게 등록하고, 그 장소가 제로페이 사용 가능한 식당인지 또는 작업하기 좋은 카페인지 함께 축적할 수 있는 내부 전용 지도를 만드는 것이다.

제품 목표는 아래와 같다.

- 네이버 지도 URL에서 장소 정보를 추출해 등록 과정을 단순하게 만든다.
- 식당과 카페를 같은 서비스 안에서 관리하되, `place_type`과 `zeropay_status`로 목적에 맞게 구분할 수 있게 만든다.
- 지도, 목록, 상세 화면을 통해 사내 구성원이 빠르게 탐색할 수 있게 만든다.
- 별점, 리뷰, 추천 기능으로 집단 지성을 누적할 수 있게 만든다.
- 회사 이메일 로그인 링크와 검색 엔진 비노출 정책으로 사내 전용 서비스를 유지한다.

## Non-Goals
이 프로젝트가 목표로 하지 않는 것은 아래와 같다.

- 일반 대중에게 공개되는 맛집/카페 서비스
- 검색 엔진 유입을 위한 공개 SEO 최적화
- 회사 외부 사용자를 위한 회원가입 또는 초대 흐름
- 결제 처리, 제로페이 결제 연동, 주문 기능
- 모바일 네이티브 앱 개발
- 관리자 전용 백오피스

## Target Users
- `@nurimedia.co.kr` 이메일을 가진 사내 구성원
- 점심 식대 사용 가능한 제로페이 가맹점을 찾고 싶은 구성원
- 작업하기 좋은 카페 정보를 공유하고 싶은 구성원
- 내가 방문했던 맛집을 공유하고 싶은 구성원

## Completed Product Criteria
프로젝트 개발이 모두 완료되었을 때 아래 기능이 갖춰져 있어야 한다.

### Authentication And Access
- 회사 이메일 로그인 링크와 같은 브라우저 90일 세션 유지가 동작한다.
- 로그인 이메일에는 바로 로그인 가능한 로그인 전용 URL이 제공된다.
- 로그인 전용 URL이 만료되었거나, 이미 사용되었거나, 새 링크 발급으로 무효화된 경우 인증 실패 화면에서 다시 로그인 흐름으로 돌아갈 수 있다.
- 비로그인 사용자는 앱과 API에 접근할 수 없다.
- 로그인 성공 후 이름이 비어 있는 사용자는 이름 입력을 완료해야 앱 메인 화면으로 진입할 수 있다.

### 장소 등록
- 사용자는 네이버 지도 URL을 입력해 `naver_place_id`와 canonical URL을 기반으로 장소를 등록할 수 있다.
- 네이버 좌표 추출 실패 시 주소 geocoding fallback을 통해 좌표를 확보하고, 최종 실패 시 저장하지 않는다.
- 사용자는 같은 장소 추가 UI 안에서 조회된 장소명과 주소를 읽기 전용으로 확인한 뒤 `place_type`, `zeropay_status`, 리뷰를 입력할 수 있다.
- 이미 review를 남긴 사용자가 같은 장소를 다시 등록하려 하면 새 review를 추가하지 않고 기존 장소 상세 화면으로 이동할 수 있다.
- 동일 `naver_place_id`는 하나의 장소로 병합된다.

### Browse And Detail
- 장소는 Kakao Map 위에서 식당/카페 구분 마커로 표시된다.
- 장소 목록에서 이름, 평균 별점, 리뷰 수를 볼 수 있고, 제로페이 가능 place에는 평균 별점 UI 높이와 같은 제로페이 로고가 표시된다.
- 장소 상세 화면에서 이름, 주소, 종류, 평균 별점, 별점 수, 내 별점 상태, 등록자, 추천 수, 리뷰를 볼 수 있다.
- 사용자는 상세 화면에서 네이버 지도 연결 버튼으로 장소를 다시 열 수 있다.
- 데스크톱과 모바일에서 각각 정의된 탐색 UX가 동작한다.

### Reviews And Recommendations
- 사용자는 리뷰를 작성할 때 별 모양 버튼으로 1점에서 5점 사이의 별점을 함께 남길 수 있다.
- 로그인 사용자는 리뷰 작성 단계에서 별점을 남길 수 있다.
- 로그인 사용자는 아직 review가 없는 장소에만 리뷰를 작성할 수 있다.
- 로그인 사용자는 추천을 추가하거나 취소할 수 있고 추천 수가 저장된다.

### Security And Operations
- 검색 엔진 차단과 최소 보안/abuse 방지 정책이 적용된다.

## Operating Constraints
- 서비스는 사내 전용이며 전체 화면과 API를 로그인 뒤에만 노출한다.
- 허용 이메일 도메인은 `@nurimedia.co.kr` 하나로 제한한다.
- 검색 엔진에 노출되지 않도록 설계한다.
- 지도는 Kakao Map을 사용한다.
- 인증과 데이터 저장은 Supabase를 사용한다.
- 배포는 Vercel을 사용한다.
- 지도 초기 중심 좌표는 `37.558721, 126.924440`이며 주소 기준점은 `서울 마포구 양화로19길 22-16`이다.
