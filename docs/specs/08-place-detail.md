# Spec FR-08: Place Detail

## Summary
place 상세 화면에 표시해야 할 정보와 상세 화면 동작을 정의한다.

## Scope
- 상세 표시 필드
- 내 별점 상태 표시
- 네이버 지도 이동 버튼

## Functional Requirements
- 상세 화면에는 이름, 주소, 종류, 평균 별점, 별점 수, 등록자, 추천 수, 리뷰를 표시한다.
- 로그인 사용자의 내 별점 상태를 보여준다.
- 네이버 지도 이동 버튼을 제공한다.
- 데스크톱과 모바일에서 동일한 정보 구조를 유지한다.

## Acceptance Criteria
- 상세 필드가 모두 표시된다.
- 내 별점 상태가 보인다.
- 네이버 지도 이동 버튼이 동작한다.

## TDD Implementation Order
1. 상세 필드 표시 테스트를 작성한다.
2. 내 별점 상태 테스트를 작성한다.
3. 네이버 지도 이동 버튼 테스트를 작성한다.
4. 구현한다.
5. 전체 테스트를 통과시킨다.

## Required Test Cases
- 평균 별점/별점 수 표시
- 등록자/추천 수/리뷰 표시
- 내 별점 상태 표시
- 네이버 지도 이동 버튼 링크

## Manual QA Checklist
- 상세에서 필요한 정보가 모두 보인다.
- 네이버 이동 버튼이 올바른 링크를 연다.

## QA Evidence
- 테스트 실행 결과
- 상세 화면 수동 검증 결과
